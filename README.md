# The-High-Noon-Shooting
Game The High Noon shooting, adalah game shooting yang dimana kalian memiliki quest atau misi untuk mengalahkan musuh kita yaitu alien. Permainan ini juga bukan hanya shooting, kita juga harus menghindari meteor meteor dalam game. Game ini juga memiliki score, dimana seberapa banyak kita tembak alien. Dalam game ini juga kita tidak asal tembak karena ada sistem "reload" untuk mengisi ulang peluru. cara bermainnya cukup simple, dengan cara pencet spasi saat menembak. Begitu juga saat ingin bergerak pencet kunci "A" dan kunci "D". Kunci "A" untuk ke kiri dan kunci "D" ke kanan.
